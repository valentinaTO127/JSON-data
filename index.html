¿Qué es una API y JSON?

API (Interfaz de Programación de Aplicaciones)

Una API es como un "mesero" que permite que diferentes aplicaciones se comuniquen entre sí:

  • Tú (cliente) haces un pedido (solicitud)
  • El mesero (API) lleva tu pedido a la cocina (servidor)
  • El mesero te trae la comida (respuesta)

JSON (JavaScript Object Notation)

JSON es un formato ligero para intercambiar datos:

  • Fácil de leer para humanos
  • Fácil de procesar para computadoras
  • Usa pares de "clave": "valor"
  • Soporta objetos {}, arrays [], strings "", números, booleanos (true/false) y null

Ejemplo de objeto JSON:

{
    "nombre": "Juan",
    "edad": 30,
    "esEstudiante": false,
    "hobbies": ["fútbol", "música", "lectura"],
    "dirección": {
        "calle": "Av. Principal",
        "ciudad": "Bogotá"
    }
}

¿Cómo funciona la notación de punto?

La notación de punto nos permite acceder a propiedades dentro de objetos JSON:

  • datos.productos - Accede a la propiedad "productos" del objeto "datos"
  • datos.productos[0] - Accede al primer elemento del array "productos"
  • datos.productos[0].nombre - Accede a la propiedad "nombre" del primer producto
  • datos.usuarios[2].dirección.ciudad - Accede a la ciudad del tercer usuario
